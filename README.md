# Discord Movie Event Scheduler

## 🚀 Installation

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

## ⚙ Commands

* `/mes [movie_url]` Create a discord event from the provided movie page

## 🎫 Supported Sites

Currently only one site is supported
| Site | URL |
| :--: | -- |
|Ingresso|<https://www.ingresso.com>|
